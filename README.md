# Employee Management System

## Overview

The Employee Management System is a Java web application developed using JSP, Servlets, JDBC, and MySQL. The system allows administrators to manage employee records efficiently through a web-based interface.

## Technologies Used

* Java
* JSP
* Servlets
* JDBC
* MySQL
* Apache Tomcat
* HTML
* CSS

## Features

* User Login Authentication
* Admin and Employee Roles
* Add Employee
* View Employees
* Edit Employee
* Delete Employee
* Pagination
* Sorting
* Session Management
* Logout Functionality
* Email Notification

## Database Tables

### Employee Table

* id
* name
* department
* salary
* email

### Users Table

* id
* username
* password
* role
